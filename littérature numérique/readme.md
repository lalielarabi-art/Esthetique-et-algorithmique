Description du projet

Pour le thesaurus j'ai choisi de changer les mots pour former un poème plutôt qu'une lettre d'amour. Donc pour l'atlier 4 j'ai continué sur cette idée de poème mais j'ai voulu le rendre interactif.
J'ai donc fait en sorte que des mots apparaissent aléatoirement en haut de l'écran et descendent lentement. Ensuite, l'utilisateur clique sur les mots qui l'inspirent pour les capturer et les ajouter à son poème en bas de l'écran.
Cela permet de créer des poèmes uniques qui ont parfois du sens ou non...
